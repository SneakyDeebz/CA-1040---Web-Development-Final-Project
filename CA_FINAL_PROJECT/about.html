About the Project

The Student Success Toolkit is a multi-page website built as the final project for an introductory Web Development course. The goal was to create a practical, student-facing website that brings together useful academic tools in one convenient location.

The website was designed from scratch using HTML, CSS, and JavaScript. No outside frameworks or libraries were used. It demonstrates core web development skills including page structure, navigation linking, external stylesheets, interactive forms, and JavaScript event handling.

What's Included

  • Home Page — Overview and navigation to all pages
  • GPA Calculator — Interactive form to calculate semester GPA using JavaScript
  • Unit Converter — Converts between common measurement units with a dropdown and JavaScript logic
  • Study Tips — Categorized productivity and academic strategies with a JavaScript filter
  • Resources — Table of recommended websites and tools for students
  • About — Project description, author info, and course details
